Creating a blogsite

A fine list of the websites and personal blogs made can be found here. You just have to download the source. After getting the source, make some changes in the default.html, which can be found in _layout folder. Make the necessary changes. Delete all the previous posts and replace the names, photos, tags etc with yours.
Now cd into the source code folder and run jekyll serve. If every thing works fine then, open your browser and type http://127.0.0.1:4000. It will run your website with the changes made. The changes done in the source code, will be updated in the browser if you run jekyll serve --w.
After finishing, you can get your website in the _site folder. Now you can upload it wherever you want or you can just push it to your git repository named username.github.com for free hosting of your pages.

Hope that helps.
